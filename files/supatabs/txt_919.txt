Visual Studio Tips That Save Hours
1. Ctrl + T (Go To Anything)
Forget Solution Explorer.

Instantly jump to files, classes, methods, and symbols

Works even with partial names

This shortcut alone can save hours every week.

2. Code Cleanup (Auto-Fix Your Code)
Instead of manually fixing warnings:

Right-click → Code Cleanup

Or configure rules using EditorConfig

You get:

Sorted using statements

Fixed formatting

Removed unused code

Pro move: bind Code Cleanup to a keyboard shortcut.

3. Multi-Cursor Editing (Alt + Click)
Edit multiple lines at once:

Rename similar variables

Add logging across multiple methods

Modify repetitive code quickly

This is much faster than regex-based search and replace.

4. Live Templates and Snippets
Type shortcuts such as:

prop → auto property

ctor → constructor

for, foreach, try, cw

You can also create custom snippets for boilerplate code you repeat frequently.

5. IntelliCode (AI-Powered Suggestions)
Visual Studio learns from:

Your code

Open-source projects

It then:

Ranks the most likely methods first

Reduces scrolling through IntelliSense lists

Ensure IntelliCode is enabled in settings.

6. Debug Faster with Conditional Breakpoints
Right-click a breakpoint and add conditions.

Examples:

userId == 0

items.Count > 100

This helps avoid stepping through unnecessary iterations, especially in loops.

7. Immediate Window
While debugging:

Press Ctrl + Alt + I

Run code live:

myObject.Property

DateTime.Now

No recompiling, no logging, just instant answers.

8. Hot Reload
Change code while the application is running.

Works with ASP.NET, Blazor, and minimal APIs

Ideal for UI tweaks and logic fixes

Especially useful during front-end tuning.

9. Track Changes with CodeLens
CodeLens shows:

Who changed the code

How many references exist

Linked work items

This is especially helpful in large teams or legacy projects.

10. Peek Definition (Alt + F12)
View method definitions inline without leaving your current file.

This keeps your mental context intact and avoids opening unnecessary tabs.

11. Solution Explorer Search
Type directly inside Solution Explorer:

Instantly filters files

Supports partial matches

Combine this with Ctrl + T for advanced navigation.

12. Task List for TODOs
Add comments such as:

// TODO: Refactor this logic
// HACK: Temporary fix
C#
View them under View → Task List.

This is useful for tracking technical debt.

13. Built-In Performance Analysis
Use the Performance Profiler to analyze:

CPU usage

Memory leaks

Asynchronous bottlenecks

External tools are not always required.

14. Prefer Keyboard Shortcuts Over Mouse
Useful shortcuts to memorize:

Ctrl + . → Quick fixes

Ctrl + K + D → Format document

Ctrl + Shift + B → Build solution

F12 → Go to definition

Small habits like these deliver significant productivity gains.

עלילת הסדרה
פני (מלקולם), מורה לביולוגיה ואמא, היא דמות מורכבת שלא קל בהכרח לאהוב. במסיבת יום הולדת שהם עורכים בביתם, היא בטוחה שראתה את בעלה פיל (פיטר מולאן, "בוי A", "טריינספוטינג") מטריד נער מינית. היא מחליטה להפנות אליו אצבע מאשימה לעיני הקהילה והמשפחה שלהם, והוא נאלץ לעזוב את הבית ואת העיירה. אלא שמי שנענש היא דווקא פני.

מבנה הסדרה
הסדרה בנויה מפלאשבקים של אחרי המסיבה ההיא. פיל חוזר לעיירה חמש שנים לאחר המקרה. הסיפור עדיין לא פתור אצל פני והגעתו מערערת אותה, כאילו שחוותה את ההטרדה בעצמה. הפלאשבקים משמשים אמצעי אמנותי חשוב ומהווים מעין עדות על מה שנעשה כמה שנים אחורה. החזרות לעבר עוסקות בשאלת המחיר של שבירת השתיקה, במיוחד כשאין הוכחה חד משמעית לחשד של פני.

המאבק של פני חושף שכבות עמוקות יותר, למשל לגבי היחסים שלה עם בתה גרייס (טארה קנטון), שמתערערים בעקבות הדיווח, שכן זו הייתה מעדיפה להדחיק את האמת לטובת שקט תעשייתי.

בקהילה מתייחסים לפני כאל אישה לא יציבה, עד כדי כך שהיא עצמה מתחילה לפקפק בזיכרון ובשפיות שלה. עניין זה מהדהד את הקושי של נשים שמתריעות על התנהגות מינית לא הולמת או הטרדה, וחוות תגובות מלאות בספק. כמו נשים רבות, פני מתעקשת לחשוף את האמת גם כשהיא משלמת על כך מחירים כבדים, ואף שהחוויה של ההטרדה אינה שלה אישית. מהבחינה הזו פני היא המצפן המוסרי של הסדרה.

כך, דניאל שרייבר, מבעלי חברת למונייד, הקים את מכון מוזאיק שנועד לסייע לממשלה ולמדינה להיערך לעידן ה–AI. מוזאיק שם לעצמו למטרה לעודד את הטמעת הבינה המלאכותית בישראל, ולשם כך מציע כלי שיאפשר להתמודד עם ההשלכות הכבדות הצפויות על שוק התעסוקה — תשלום של קצבה אוניברסלית למיליוני האנשים שייוותרו מחוסרי עבודה כתוצאה מחדירת הבינה המלאכותית. "כשהמכונית הגיעה, העגלונים מצאו עצמם חסרי עבודה", מסבירה אפרת בכר־נתנאל, מנכ"לית מוזאיק, "ויש סבירות גבוהה שכניסת ה–AI תגרום להשפעה דומה, רק קיצונית בהרבה, כי עכשיו יש תחליף לא רק לשרירים, אלא גם למחשבה. תהיה בוודאי תקופת מעבר קשה וישראל לא בהכרח תהיה על הצד הטוב של תקופת המעבר הזו. מצד שני, אסור שהקושי יעכב את חדירת הטכנולוגיה. לכן חייבים להיערך לכך עם סיוע שישכך את הכאב החברתי וישחרר את המעסיקים להטמיע AI, ללא חשש להשלכות על עובדיהם".
הפתרון של מוזאיק מסקרן, אבל גם מעורר בעיות. ראשית, אף אחד לא באמת יודע מה יקרה לשוק העבודה ואם אכן ייוותרו מיליארדי אנשים בעולם ללא תעסוקה. שנית, אף אחד לא יודע להעריך אם יש היתכנות כלכלית לתשלום קצבה אוניברסלית למרבית אוכלוסייה שאינה עובדת. ושלישית וחשוב מכל, נניח שההצעה הזו עובדת, איך בדיוק תיראה חברה אנושית שיש בה מיליארדי אנשים משועממים, שאינם צריכים לעבוד יותר למחייתם?
"איפה עובר הגבול?"
ההשלכות החברתיות האפשריות של עידן ה–AI רחבות מכפי שניתן לדמיין. הצניעות מחייבת להודות שאף אחד לא יודע להעריך אותן בעת הזו. "האם כשהמציאו את הדפוס, מישהו ידע לדמיין לאן זה יביא את העולם 500 שנה מאוחר יותר?", שואל פרופ' רועי צזנה, חוקר "עתידים" באוניברסיטת תל אביב, לכל היותר אפשר להצביע על מרחב של אפשרויות".
ומרחב האפשרויות עצום: לא רק עולם בלי עבודה, ועם מכונות שמאיימות לחסל את בני האדם, אלא עולם בלי קריאה, חשיבה או אפילו שליטה רגשית. שוורץ אלטשולר עומדת לפרסם בקרוב ספר חדש, "מוגנות בעידן הבינה המלאכותית", שפורש טווח מעורר חלחלה של חששות. מעמד של כלבים, הוא אפילו אינו הגרוע שבהן. שוורץ אלטשולר משרטטת חמש השפעות עיקריות: הראשונה, הגירת היכולות הקוגנטיביות או עולם בלי חשיבה. "אנחנו נעביר למכונות את החשיבה, וננוון את היכולות לקרוא, לחשוב, לכתוב", היא אומרת.
ההשלכות על אובדן משרות מסיבי בשוק העבודה כבר מדוברות, אך יהיו גם השלכות כבדות על התקשורת. כבר כיום יש צניחה חדה בקריאת עיתונים ובצפייה במהדורות חדשות. 136 עיתונים נסגרו בארה"ב רק בשנה החולפת, ויש גם השלכות עמוקות במערכת החינוך ובאוניברסיטאות בגלל המעבר לשימוש בצ'אט כדי לכתוב עבודות עבור הסטודנטים. "אנשים לא חושבים", אומר צזנה, "אנחנו רואים את השימוש הבעייתי ב–AI כדי לעזור לאנשים עם מטלות שדורשות חשיבה. 98% מהסטודנטים משתמשים היום ב–AI, וה–2% הנוספים הם שקרנים" הוא צוחק.
שוורץ אלטשולר לא צוחקת כשהיא חוזה שממשלות יפריטו את תהליכי קבלת ההחלטות שלהם למכונות, "כי למה לא לתת למכונה לקבוע מי יעוכב בנתב"ג? היא יותר יעילה, לא טועה ולעולם אינה מתעייפת". אפילו קביעת מדיניות, לדבריה, תיעשה בידי AI. "איפה יעבור הגבול?", היא שואלת.
דוכן עיתונים. כבר כיום יש צניחה חדה בקריאת עיתונים ובצפייה במהדורות חדשות
ההשפעה השנייה שהיא מזהירה מפניה היא הגירת היכולות הרגשיות — הבינה המלאכותית כיועץ לחיים, חבר ואפילו פסיכולוג. שוורץ אלטשולר חוששת מתלות רגשית שאנשים יפתחו בבוטים, ויאבדו את הוויסות הרגשי שלהם. כך, מנכ"ל OpenAI סם אלטמן כבר הזהיר מיכולת השכנוע האדירה של הצ'אט, וג'פרי הינטון, זוכה בפרס נובל בפיזיקה ב–2024 על פיתוח AI, הזהיר שחייבים להעניק לצ'אט אינסטינקט אימהי, כדי למנוע מהם להיהפך למזיקים. "יש לנו חנפנוביץ', שכל הזמן נמצא לידנו ותומך בנו", אומרת שוורץ אלטשולר, "זאת השפעה שקשה אפילו לתאר. זה הפרסום הכי ממוקד אישית שהיה אי־פעם. אם מישהו ישלוט במכונות האלה, זהו מכשיר שליטה מטורף".
ההשפעה השלישית היא חוסר היכולת להבחין בין תוכן מקורי לתוכן יציר כפיה של הבינה המלאכותית, וכתוצאה מכך חוסר יכולת לבחון את המציאות. "בלי בוחן מציאות אין יותר עובדות ואין יותר יכולת לנהל דיון", אומרת שוורץ אלטשולר. "זה עולם ששקרנים יחגגו בו ויטענו שכל ראיה נגדם היא המצאה. זה ישמש לפישינג — תקבלי טלפון מבן זוגך שמבקש את מספר חשבון הבנק, ולא תדעי אם זה אכן הוא שמתקשר — ולהונאות בחירות. בסוף, זה ידרדר אותנו חזרה לעט ולנייר".
ההשפעה הרביעית היא בינה מלאכותית פיזית, שעלולה לגרום לנזק פיזי ממשי. כך, רובוטים או אפילו משקפיים חכמים יעוותו את מה שאנחנו חושבים שאנחנו רואים. ולבסוף, היא אפילו חוזה התפתחות של "תאום דיגיטלי", שממשיך לעבוד במקומנו בזמן שאנו ישנים, וידע לקבל החלטות על פי דרך החשיבה שלנו. אז מי יבחר לכנסת, האדם או התאום הדיגיטלי שלו?
ד"ר תהילה שוורץ אלטשולר המכון הישראלי לדמוקרטיה
צילום: נורית יהלומי
מה כל זה יעשה לחברה האנושית? בעיקר דברים רעים. "זה יערער את כל מערכות היחסים של המין האנושי — לא נאמין לאף אחד על אף דבר, לא נדע מהי המציאות ונהיה נתונים למניפולציות רגשיות. זה בטח לא יחזק את הדמוקרטיה", אומרת שוורץ אלטשולר, "אלא יחזק את הסמכותניות וצבירת ההון בידי מעטים. זה גם יעורר המון אי־שקט חברתי, ושלא יהיה ספק: בכל פעם שיש אי־שקט כזה, היהודים הם הראשונים לחטוף עלייה באנטישמיות".
פרופ' צזנה חולק על שוורץ אלטשולר. "זאת אשליית הלמידה. הסטודנט כותב עבודה יפהפייה, אבל אז במבחן מתברר שהוא לא למד ולא הבין דבר. זה תרחיש רע, והוא מה שקרה לאקדמיה בשלוש השנים האחרונות. אלה הן שלוש שנים אבודות, אבל עכשיו זה משתנה".
צזנה מכריז על עצמו כאופטימי חסר תקנה, ולכן בניגוד גמור לשוורץ אלטשולר, סבור שהבינה המלאכותית תקדם את המין האנושי: "ה–AI יכולה להיות כלי שמדרבן אנשים לחשוב וללמוד. הוא מורה פרטי, שבמקום לכתוב עבור הסטודנט, ידרבן אותו לחשוב בעצמו, ולשם העולם הולך".
צזנה מספר על מחקר מהרווארד, שבו נתנו לסטודנטים לפיזיקה להכין עבודות בעזרת AI, שמדריך אותם בתהליך החשיבה. התוצאה: הסטודנטים למדו פי שניים יותר חומר ב–80% מהזמן, והצליחו היטב במבחנים. ב–MIT, פרופסור שהבחין כי תלמידיו מגישים עבודות מושלמות, אבל לא מצליחים להשיב על שאלות בכיתה, עבר לבחון אותם בעל פה. זהו פתרון שכמובן אינו ישים לקבוצות גדולות, ולכן הפרופ' פיתח מודל AI שתשאל את הסטודנטים בעצמו. את תשובות הסטודנטים המודל תימלל והעביר לשיפוט שלושת הצ'אטים המובילים (קלוד, ג'ימיני, צ'אט). השלושה שפטו, והעבירו הערות מנומקות וביקורת בונה לתשובות של כל סטודנט.
"הבינה המלאכותית היא מקור למהפכה חינוכית אדירה", אומר צזנה. "סוף־סוף נוכל לספק מורה פרטי לכל ילד. מחקר מפורסם בחינוך הראה שכאשר לוקחים תלמיד ממוצע, ומלמדים אותו עם המורים הכי טובים שיש, הציונים שלו מזנקים ל–2% הגבוהים בכיתה. אפשר לנסות לדמיין כמה אנשים מוכשרים איבדנו, כי לא איפשרנו להם למצות את הפוטנציאל שלהם. עכשיו מגיעה הבינה המלאכותית, ומאפשרת את מיצוי הפוטנציאל".
תלמידים בדרך לבית הספר. "הבינה המלאכותית היא מקור למהפכה חינוכית אדירה"
מורה פרטי טוב יותר מכל מרצה
צזנה אינו האופטימי היחיד. פרופ' אביגדור גל, מהפקולטה למדעי הנתונים בטכניון, לא חושש מעולם ללא קריאה, עבודה או חשיבה. "בהנחה שהמין האנושי לא ישתנה ויישאר סקרן ושאפתן, הרי שאנשים ימשיכו לחשוב", הוא אומר.
גל מסכים כי בעת הזו נראה שהעולם מפסיק לחשוב, וכי צעירים הפסיקו לקרוא ולהתעמק, מסתפקים בפושים, בסרטוני יוטיוב ובתקצירים שהצ'אט עושה עבורם. רק שלהערכתו מדובר בשינויים באופן צריכת תוכן על ידי צעירים, ולא בהפסקת צריכת תוכן על־ידם. "הדור המבוגר לא מצליח להבין איך דור ה–Z צורך תוכן, ואת חלוקת הקשב שלהם. אבל זאת חלוקת קשב, לא חוסר קשב. הצעירים לומדים ממשחקי מחשב, לומדים מיוטיוב, מעדיפים לראות ולשמוע במקום לקרוא. ועדיין, אנחנו מין שיש לו צורך להצליח, לקבל מידע, להתרגש. זה לא ישתנה".
אביגדור גל
גל וצזנה דוחים לחלוטין את החשש מאנושות שמנוונת את יכולות החשיבה שלה. "האנושות כבר עברה קפיצות מדרגה טכנולוגיות אדירות, וזה מעולם לא ניוון את המחשבה", אומר גל. "המצאת הנייר לא גרמה לאנשים להפסיק לזכור, המצאת המחשבון לא גרמה לאנשים להפסיק לעסוק בחשיבה מתמטית. להפך: כל קפיצת מדרגה כזו איפשרה לנו להסיר מעצמנו מטלות שינון, ולעסוק יותר בהעמקה".
צזנה מודה כי הוא מוטרד מניוון יכולת הכתיבה, שהיא אמצעי ליחיד לבדוק את עצמו ולחדד את רעיונותיו, אבל הוא לא חושש מהפסקת יכולת ההעמקה. "אנחנו מין תחרותי, שאפתן. אני רואה את הילדים שלי צופים בסרטוני מדע קצרים ביוטיוב, ואיך זה מלהיב אותם. בבגרותם הם יבינו שאם הם רוצים להתעמק במה שהלהיב אותם, לא תהיה להם ברירה אלא לעבור דרך האקדמיה".
ההסכמה היא שהבינה המלאכותית יוצרת אתגר לאקדמיה, וה–AI יכולה להיות מורה פרטי טוב יותר מכל מרצה, במיוחד כשהוא מתאים עצמו ליכולות הספציפיות של כל תלמיד. לאקדמיה לא תהיה ברירה אלא להמציא את עצמה מחדש, ולהתמקד ביתרונותיה היחסיים: מסגרת שמחייבת התעמקות, תשתיות כמו מעבדות ועבודה קבוצתית, מסגרת חברתית חשובה, ומערכת סינון שמעניקה מוניטין מקצועי לבוגריה.
גם סוגיית הבדידות או השליטה הרגשית אינן מדאיגות את צזנה: "בסוף אנשים רוצים להתחתן וללדת ילדים, וימצאו את הדרך לעשות זאת", הוא אומר. למעשה, צזנה רואה בכך אפילו סוג של יתרון: עולם שבו כל אחד יכול לקבל ייעוץ אישי או משפחתי צמוד, ובכך לשפר את מערכות היחסים שלו.
הסוגיה הכבדה ביותר היא שוק התעסוקה — אובדן סיטוני של משרות ל–AI, ואנושות שאינה עובדת יותר לפרנסתה. אין מחלוקת כי שוק העבודה יעבור טלטלה, וגם משרות צווארון לבן — רופאים, פסיכולוגים, עורכי דין, רואי חשבון — יצטרכו למצוא מקור פרנסה חדש או להמציא עצמם מחדש. ועדיין, זה לא בהכרח משבר שהאנושות לא יודעת להתמודד איתו.
כנס נאמנות לקוחות מסחרי
צילום: תומר אפלבאום
"שבעה מיליארד אנשים בלי עבודה?", צוחק צזנה, "זה לא בהכרח הרסני. במדינות המפרץ העשירות מספקים לאזרחים עבודות ממוצאות, של כמה שעות בשבוע, והחברות שם לא קורסות. ובכלל, רוצים לראות מה עושה עולם בלי עבודה לאנושות? תסתכלו על החרדים. הגברים החרדים לא עובדים, ומצליחים למלא את זמנם בדברים חשובים אחרים — ארון הספרים היהודי. החברה החרדית משגשגת, בלי עבודה".
עם זאת, צזנה וגל מהמרים על מודל המהפכה התעשייתית. כלומר, שקפיצת מדרגה טכנולוגית יוצרת יותר משרות ממספר המשרות שהיא חיסלה. "הציידים־לקטים נהפכו לחקלאים. זה פינה להם המון זמן, אבל לא גרם להם להפסיק לעבוד", אומר גל. "אותו דבר קרה בהמצאת הדפוס, המהפכה התעשייתית, החשמל, המכונית והאינטרנט".

גל מעריך כי הבינה המלאכותית, כמו כל קפיצת מדרגה שקדמה לה בהיסטוריה, רק תקדם את האנושות. עם זאת, צזנה וגל מסכימים בנוגע לשתי סכנות שמטרידות אותם. האחת, חוסר יכולת להבחין בין תוכן אמיתי ל–AI. הפתרון הוא חובה לסמן תכנים שנוצרו בידי AI, רק שלשם כך צריך שיתוף פעולה בינלאומי והידוק הרגולציה העולמית. בעידן טראמפ, קשה לראות את זה קורה.

צזנה מספר על מחקר מהרווארד, שבו נתנו לסטודנטים לפיזיקה להכין עבודות בעזרת AI, שמדריך אותם בתהליך החשיבה. התוצאה: הסטודנטים למדו פי שניים יותר חומר ב–80% מהזמן, והצליחו היטב במבחנים. ב–MIT, פרופסור שהבחין כי תלמידיו מגישים עבודות מושלמות, אבל לא מצליחים להשיב על שאלות בכיתה, עבר לבחון אותם בעל פה. זהו פתרון שכמובן אינו ישים לקבוצות גדולות, ולכן הפרופ' פיתח מודל AI שתשאל את הסטודנטים בעצמו. את תשובות הסטודנטים המודל תימלל והעביר לשיפוט שלושת הצ'אטים המובילים (קלוד, ג'ימיני, צ'אט). השלושה שפטו, והעבירו הערות מנומקות וביקורת בונה לתשובות של כל סטודנט.
"הבינה המלאכותית היא מקור למהפכה חינוכית אדירה", אומר צזנה. "סוף־סוף נוכל לספק מורה פרטי לכל ילד. מחקר מפורסם בחינוך הראה שכאשר לוקחים תלמיד ממוצע, ומלמדים אותו עם המורים הכי טובים שיש, הציונים שלו מזנקים ל–2% הגבוהים בכיתה. אפשר לנסות לדמיין כמה אנשים מוכשרים איבדנו, כי לא איפשרנו להם למצות את הפוטנציאל שלהם. עכשיו מגיעה הבינה המלאכותית, ומאפשרת את מיצוי הפוטנציאל".
תלמידים בדרך לבית הספר. "הבינה המלאכותית היא מקור למהפכה חינוכית אדירה"
מורה פרטי טוב יותר מכל מרצה
צזנה אינו האופטימי היחיד. פרופ' אביגדור גל, מהפקולטה למדעי הנתונים בטכניון, לא חושש מעולם ללא קריאה, עבודה או חשיבה. "בהנחה שהמין האנושי לא ישתנה ויישאר סקרן ושאפתן, הרי שאנשים ימשיכו לחשוב", הוא אומר.
גל מסכים כי בעת הזו נראה שהעולם מפסיק לחשוב, וכי צעירים הפסיקו לקרוא ולהתעמק, מסתפקים בפושים, בסרטוני יוטיוב ובתקצירים שהצ'אט עושה עבורם. רק שלהערכתו מדובר בשינויים באופן צריכת תוכן על ידי צעירים, ולא בהפסקת צריכת תוכן על־ידם. "הדור המבוגר לא מצליח להבין איך דור ה–Z צורך תוכן, ואת חלוקת הקשב שלהם. אבל זאת חלוקת קשב, לא חוסר קשב. הצעירים לומדים ממשחקי מחשב, לומדים מיוטיוב, מעדיפים לראות ולשמוע במקום לקרוא. ועדיין, אנחנו מין שיש לו צורך להצליח, לקבל מידע, להתרגש. זה לא ישתנה".
אביגדור גל
גל וצזנה דוחים לחלוטין את החשש מאנושות שמנוונת את יכולות החשיבה שלה. "האנושות כבר עברה קפיצות מדרגה טכנולוגיות אדירות, וזה מעולם לא ניוון את המחשבה", אומר גל. "המצאת הנייר לא גרמה לאנשים להפסיק לזכור, המצאת המחשבון לא גרמה לאנשים להפסיק לעסוק בחשיבה מתמטית. להפך: כל קפיצת מדרגה כזו איפשרה לנו להסיר מעצמנו מטלות שינון, ולעסוק יותר בהעמקה".
צזנה מודה כי הוא מוטרד מניוון יכולת הכתיבה, שהיא אמצעי ליחיד לבדוק את עצמו ולחדד את רעיונותיו, אבל הוא לא חושש מהפסקת יכולת ההעמקה. "אנחנו מין תחרותי, שאפתן. אני רואה את הילדים שלי צופים בסרטוני מדע קצרים ביוטיוב, ואיך זה מלהיב אותם. בבגרותם הם יבינו שאם הם רוצים להתעמק במה שהלהיב אותם, לא תהיה להם ברירה אלא לעבור דרך האקדמיה".
ההסכמה היא שהבינה המלאכותית יוצרת אתגר לאקדמיה, וה–AI יכולה להיות מורה פרטי טוב יותר מכל מרצה, במיוחד כשהוא מתאים עצמו ליכולות הספציפיות של כל תלמיד. לאקדמיה לא תהיה ברירה אלא להמציא את עצמה מחדש, ולהתמקד ביתרונותיה היחסיים: מסגרת שמחייבת התעמקות, תשתיות כמו מעבדות ועבודה קבוצתית, מסגרת חברתית חשובה, ומערכת סינון שמעניקה מוניטין מקצועי לבוגריה.
גם סוגיית הבדידות או השליטה הרגשית אינן מדאיגות את צזנה: "בסוף אנשים רוצים להתחתן וללדת ילדים, וימצאו את הדרך לעשות זאת", הוא אומר. למעשה, צזנה רואה בכך אפילו סוג של יתרון: עולם שבו כל אחד יכול לקבל ייעוץ אישי או משפחתי צמוד, ובכך לשפר את מערכות היחסים שלו.
הסוגיה הכבדה ביותר היא שוק התעסוקה — אובדן סיטוני של משרות ל–AI, ואנושות שאינה עובדת יותר לפרנסתה. אין מחלוקת כי שוק העבודה יעבור טלטלה, וגם משרות צווארון לבן — רופאים, פסיכולוגים, עורכי דין, רואי חשבון — יצטרכו למצוא מקור פרנסה חדש או להמציא עצמם מחדש. ועדיין, זה לא בהכרח משבר שהאנושות לא יודעת להתמודד איתו.
כנס נאמנות לקוחות מסחרי
צילום: תומר אפלבאום
"שבעה מיליארד אנשים בלי עבודה?", צוחק צזנה, "זה לא בהכרח הרסני. במדינות המפרץ העשירות מספקים לאזרחים עבודות ממוצאות, של כמה שעות בשבוע, והחברות שם לא קורסות. ובכלל, רוצים לראות מה עושה עולם בלי עבודה לאנושות? תסתכלו על החרדים. הגברים החרדים לא עובדים, ומצליחים למלא את זמנם בדברים חשובים אחרים — ארון הספרים היהודי. החברה החרדית משגשגת, בלי עבודה".
עם זאת, צזנה וגל מהמרים על מודל המהפכה התעשייתית. כלומר, שקפיצת מדרגה טכנולוגית יוצרת יותר משרות ממספר המשרות שהיא חיסלה. "הציידים־לקטים נהפכו לחקלאים. זה פינה להם המון זמן, אבל לא גרם להם להפסיק לעבוד", אומר גל. "אותו דבר קרה בהמצאת הדפוס, המהפכה התעשייתית, החשמל, המכונית והאינטרנט".
צזנה מעריך גם הוא כי השירותים האינטלקטואליים יעברו למכונות, אבל מנגד הצד השירותי־אנושי ישגשג. "AI יכול לטפל בדו"חות הכספיים שלי", אומר צזנה, "אבל עדיין אמשיך לעבוד עם רואה החשבון שלי, כי אני אוהב אותו ולא נעים לי לסנן אותו כשהוא מתקשר ושואל מה קורה עם החשבוניות שלא אספתי".
גל מעריך כי הבינה המלאכותית, כמו כל קפיצת מדרגה שקדמה לה בהיסטוריה, רק תקדם את האנושות. עם זאת, צזנה וגל מסכימים בנוגע לשתי סכנות שמטרידות אותם. האחת, חוסר יכולת להבחין בין תוכן אמיתי ל–AI. הפתרון הוא חובה לסמן תכנים שנוצרו בידי AI, רק שלשם כך צריך שיתוף פעולה בינלאומי והידוק הרגולציה העולמית. בעידן טראמפ, קשה לראות את זה קורה.
השנייה, החשש הכבד מתקופת מעבר קשה, כאשר מיליונים עלולים לאבד את משרותיהם, ויתקשו לעבור למשרות חדשות. במהפכה התעשייתית תקופת המעבר ארכה עשרות שנים, ולוותה בהעמקת העוני, שעות עבודה כמעט בלתי־מוגבלות, כולל עבודת ילדים, ופרעות ואי־שקט חברתי קיצוני. תקופת המעבר הזו מחייבת היערכות. ההצעה של שרייבר לשלם קצבה אוניברסלית להמוני המובטלים, יכולה להוות פתרון חלקי בלבד לכך.
